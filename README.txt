MyCalculator WPF Application is a simple calculator that can perform simple operations
Addition, subtraction, multiplication and division on two entered numbers and the result
is displayed in the results label.

MyCalculator is developed with the following technologies:
	1.WPF application
	2.MVVM pattern
	3.Caliburn.Micro framework is used
	4.IoC Container is implemented
	5.Application has one dialog window
	6.Serilog Logger is implemented 

user manual:
1.Use the application by entering two numbers and clicking any button 
(Addition, subtraction, multiplication or division) to perform mathematical calculations.
2.The Clear button also with user aggregation deletes the two numbers and result for using 
the application for different numbers.
